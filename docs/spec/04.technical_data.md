# 技術・データ管理

## 1. データ管理 (Data Persistence)
- **DataManager**: `GameWorks OAK Common Library` の `DataManager` を直接参照。LocalStorage の読み書きを一元管理。
- **標準データ構造**: セーブデータは `{ v: version, d: data }` のラッパー形式で永続化し、管理用メタデータと実データを分離。
- **共有リソースの利用**: ライブラリ本体は GameWorks OAK ポータルが提供する URL から直接インポートされ、プロジェクトを跨いだロジックの同一性を保証。

## 2. 技術スタック
- **Core**: HTML5, CSS3 (Vanilla), Vanilla JavaScript
- **Module System**: ES Modules (ESM)
- **Build Tool**: Vite
- **Sound**: Web Audio API (Manual Synthesis & Scheduling)
- **Testing**: Vitest (jsdom environment)
- **Library**: GameWorks OAK Common Library (Remote ESM Import)
