# Skill: start-dev

## 目的
開発作業を開始するための準備を行う。

## 手順
1. 共通仕様（`C:\Users\tioak\Documents\Games\Projects\GameWorksOAK\GameWorksOAK\docs\common`）を確認し、プロジェクト全体の標準やガイドラインを把握する。
2. `docs/` フォルダー配下のドキュメント（`backlog.md`, `dev_handover.md`, `local_RULES.md` 等）を確認し、現状を把握する。
3. 今回開発する内容を提案する。
    - `docs/dev_handover.md` がある場合は、そこに記載されている「次のチャットへの宿題」を最優先で検討する。
4. ユーザーと開発内容について合意を得る。
5. 適切な名前でブランチを作成する。
6. 作成したブランチ上で、合意内容に基づき開発計画を `docs/dev_implementation_plan.md` として作成（または更新）する。
7. 開発を開始する。
